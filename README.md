# Lindberg_etal_2026_PNAS
Raw data and code used to produce the plots for the article: "The temperature dependence of amyloid β solubility reveals the hydrophobic effect as the main driving force for fibril formation"

## To run the code yourself:
1. Install a python virtual environment manager, e.g. conda. I recommend [uv](https://docs.astral.sh/uv/getting-started/installation/)
2. `uv venv ab-temp-sol`
3. `source ab-temp-sol/bin/activate.<your shell>`
4. `uv pip install jupyterlab shapely polars matplotlib scikit-learn`
5. `jupyter lab`
